# 超级包菜 隐私政策

**发布日期：2023年11月19日**
**生效日期：2023年11月19日**

欢迎使用 超级包菜！

超级包菜（以下简称“本应用”或“我们”）深知个人信息对您的重要性，并会尽全力保护您的个人信息安全可靠。我们致力于维持您对我们的信任，恪守以下原则，保护您的个人信息：权责一致原则、目的明确原则、选择同意原则、最少够用原则、确保安全原则、主体参与原则、公开透明原则等。

请在使用本应用前，仔细阅读并理解本《隐私政策》。

## 1. 我们如何收集和使用您的个人信息

为实现本应用的核心功能，我们可能需要收集和使用您的以下个人信息：

- **统一身份认证信息**：当您登录时，您需要提供您的学号和密码以完成身份验证。我们仅在本地设备上通过加密方式存储您的凭据，用于自动登录，不会上传至任何服务器。
- **教务系统信息**：为了向您提供课表、成绩、考试安排等查询服务，本应用会访问学校的教务系统，并将查询结果展示给您。这些信息仅用于展示，我们不会存储或分析这些数据。
- **设备信息**：为了适配不同的设备并进行问题排查，我们可能会收集您的设备型号、操作系统版本等基本信息。
- **网络信息**：为保证服务的正常运行，我们会访问您的网络状态信息。

## 2. 我们如何存储您的个人信息

- **存储位置**：您的个人信息（如登录凭据）将通过加密方式存储在您的设备本地，我们不会将这些信息上传到服务器。
- **存储期限**：我们仅在您使用本应用期间保留您的信息。当您卸载本应用或手动清除缓存时，相关信息将被删除。

## 3. 我们如何共享、转让、公开披露您的个人信息

我们不会与任何公司、组织和个人共享、转让或公开披露您的个人信息，但以下情况除外：
- 在获取明确同意的情况下；
- 根据适用的法律法规、法律程序的要求、强制性的行政或司法要求所必须的情况下。

## 4. 我们如何保护您的个人信息安全

我们已使用符合业界标准的安全防护措施保护您提供的个人信息，防止数据遭到未经授权的访问、公开披露、使用、修改、损坏或丢失。例如，我们通过 HTTPS 协议保障数据传输的加密安全，并通过加密算法存储您的登录凭据。

## 5. 您的权利

您有权访问、更正、删除您的个人信息。您可以通过以下方式实现：
- **清除登录信息**：在应用的设置页面，您可以随时退出登录并清除本地存储的账号信息。
- **卸载应用**：卸载本应用将删除所有本地数据。

## 6. 本政策如何更新

我们可能会适时对本隐私政策进行调整或变更。未经您明确同意，我们不会削减您按照本隐私政策所应享有的权利。任何更新都会在本页面上发布，我们建议您定期查看。
